1.分词（tokenize）
> 文字 → 整数
神经网络只能算数字，不能算文字。所以第一个问题是：怎么把"从前有一只小猫"变成数字？
BPE 的核心思想很简单：从单字开始，不断合并最频繁的相邻对。比如"从前"经常一起出现，就合并为一个 token。最终得到一个 32000 大小的"子词词表"。
分词本身不是任何模型的专属操作——它是文本进入任何神经网络的"大门"。Transformer 要分词，Mamba 要分词，就连你用 CNN 做文本分类也要分词。
2词嵌入（Embedding）
> 整数 → 向量
整数之间没有"意义上的距离"——数字 1823 和 1824 在语义上可能毫无关系。
Embedding 就是一张查找表：
```python
# 我们代码中的 model.py 第40行
self.embedding = nn.Embedding(vocab_size=32000, d_model=1024)
```
